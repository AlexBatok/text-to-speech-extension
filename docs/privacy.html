Privacy Policy

Text to Speech — Read Aloud Any Page

Data Collection

This extension does not collect, store, or transmit any personal data. No analytics, no tracking, no telemetry.

Text Processing

All text extraction and speech synthesis occur locally on your device using the browser's built-in text-to-speech engine. No text content is sent to any external server.

Network Requests

This extension makes zero network requests. It operates entirely offline and does not communicate with any external services.

Storage

The extension stores your preferences (selected voice, speed, pitch, volume) locally in your browser using the Chrome Storage API. This data never leaves your device.

Permissions

  • activeTab — Access the current tab to extract visible text when you click the extension
  • contextMenus — Add a "Read aloud" option to the right-click menu for selected text
  • storage — Save your voice and playback preferences locally
  • tts — Use the browser's built-in text-to-speech engine
  • scripting — Inject the content script to extract text and show the player widget

Third Parties

This extension does not use any third-party services, libraries, or analytics tools. All code is bundled within the extension.

Changes

If this policy changes, updates will be posted on this page. The extension itself will always maintain zero data collection.
